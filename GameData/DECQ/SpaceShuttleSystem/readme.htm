Space Shuttle System (SSS)

Space Shuttle System is a high quality, high fidelity Space Shuttle mod for Kerbal Space Program

By zer0Kerbal, originally by DECQ with Olympic1 with contributions by Radar

adopted with express permission and brought to you by KerbSimpleCo

Preamble by DECQ

Hello all ! This addon adds details to build Space Shuttle in KSP. First of all it is worth noting that the entire system is taken from the add-on for Orbiter2010 (Where is the original space shuttle) With the modernization of the name of SSU 3.0, with the approval of the person Urwumpe, and released under the license to GPL version 2. Since the original work, the use of texture created by Wolf for Orbiter2010 SSU 3.0

Might need to use a VPN to access
